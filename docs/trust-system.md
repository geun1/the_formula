# 매너온도 & 등급 시스템 설계

> 관리 파일: `src/lib/trust.ts`

---

## 설계 원칙

- **등급 조건** — "이런 활동을 해본 사람에게 이 레벨을 줄 수 있는가" 최소 자격
- **온도** — "얼마나 열심히, 반응을 얼마나 받았는가" 활동 강도
- **두 조건이 모두 충족될 때 레벨업**

---

## 등급 (Tier)

단계형 진입. 이전 등급 조건을 충족해야 다음 등급에 진입할 수 있다.

| 등급 | 온도 하한 | 진입 조건 (누적) |
|------|----------|----------------|
| 🌱 새싹 (sprout) | 0점 | 없음 |
| 🌿 기여자 (contributor) | 15점 | 공식 1개 이상 작성 |
| 🔥 활동가 (activist) | 38점 | 기여자 조건 + 완주 1회 이상 |
| 🛠 빌더 (builder) | 65점 | 활동가 조건 + 완주 3회 이상 |
| 👑 AX마스터 (master) | 82점 | 빌더 조건 + 완주 5회 + 검증 공식 3개 + 하트 50개 |

> "단계형": contributor 없이 activist가 될 수 없고, activist 없이 builder가 될 수 없다.

---

## 온도 계산

```
온도 = 100 × raw / (raw + 30)
```

감속곡선(SATURATION_K=30). raw가 올라갈수록 온도 상승 폭이 줄어든다. 100점에 점근.

### 신호별 가중치 (WEIGHTS)

| 신호 | 가중치 | 비고 |
|------|--------|------|
| 모임·스터디 완주 | 20점/회 | 가장 강한 신호 |
| 검증된 공식 | 12점/개 | |
| 공식 작성 | 4점/개 | |
| 공식 저장받음 | 0.5점/회 | |
| 받은 하트 | 0.3점/개 | |
| 좋아요 받기 | 0.3점/개 | |
| 댓글 받기 | 0.5점/개 | |
| 온보딩 완료 | 10점 | 기본 온도 |

### 점수 잠금

등급 조건을 충족하지 못하면 해당 등급의 시작 온도 미만으로 점수 상한을 설정한다.  
예: 공식이 없으면(contributor 조건 미충족) 온도가 아무리 높아도 14.9점 이하로 제한.
