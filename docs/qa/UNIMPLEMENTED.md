# 미구현 / 미완 기능 백로그

구현된 페이지 안에서 발견한 "비어있거나 / 자리표시자거나 / 절반만 된" 기능. 루프를 돌며 보완 대상으로 끌어온다.

우선순위: 🔴 핵심 흐름 차단 · 🟡 체감 품질 · 🟢 보완하면 좋음
상태: TODO · DOING · DONE(구현+검증)

## 식별 휴리스틱 (탐색 중 이런 걸 잡는다)
- 클릭되는데 아무 일도 안 일어나는 버튼/링크 (no-op)
- "준비중 / TODO / Coming soon / 미구현" 텍스트, 빈 섹션, 더미 데이터
- 폼은 있는데 제출 핸들러가 없거나 서버 액션이 비어있음
- 권한 분기 중 한 갈래만 구현되고 나머지는 미처리
- 목록은 있는데 상세/생성/수정/삭제 중 일부 라우트 부재 (CRUD 구멍)
- 에러/빈/로딩 상태가 없어 깨져 보이는 화면
- API는 있는데 UI에서 호출 안 함 (또는 그 반대)
